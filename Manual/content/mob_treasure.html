A treasure refers to any main collectible that the Pikmin can deliver back to the ship. These are the ship parts in Pikmin, the treasures in Pikmin 2, and the fruits in Pikmin 3.

Examples of treasure types include the Main Engine, the Courage Reactor, and the Sunseed Berry.

To create a new type of treasure, please follow the making object types tutorial.

Overview

A treasure type object simply has logic to be carried, and to give a different value of points. The only other important parts of the treasure are their radius and weight.

Treasure type objects are hardcoded in the engine to sit there, be carried, be delivered to a ship, and give out some number of points.

Properties

Besides the global object type properties, a treasure type has the following properties.

Mandatory

None.

Recommended

  • points: How many points the treasure is worth when salvaged. Defaults to 10.

Optional

None.

Animations

A treasure type needs to have the following animations:

  • idling: Idling; used for everything.

Script

Treasure types have hardcoded scripts, so the only thing you can do is enhance them.

Some script variables control special behavior, like the treasure's starting team. To know which ones exist, you can go to the area editor, select a treasure, and read the tooltip of each variable.
